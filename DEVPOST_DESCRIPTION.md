# AgentWatch - AI Agent Cost Monitor & Guardrail Dashboard

## Problem
AI agents can silently burn through budgets. Product managers have no easy way to monitor costs across multiple agents in real-time or set automatic guardrails to prevent runaway spending. One runaway agent can cost thousands of dollars before anyone notices.

## Solution
AgentWatch gives product managers a single dashboard to track all AI agent costs, set per-agent budgets, receive smart alerts when approaching limits, and auto-stop agents that exceed their budgets - preventing costly overruns before they happen.

## Key Features

### Real-time Cost Tracking
- Live cost updates across all AI agents
- 7-day, 30-day, and 90-day cost trend charts
- Per-agent cost breakdown with model information

### Budget Guardrails
- Set daily budgets for each agent individually
- Visual progress bars showing spending vs limits
- Color-coded alerts (green/yellow/red) for budget status
- Automatic agent stop when budget exceeded

### Smart Alert System
- Instant alerts when agents approach budget limits (80% threshold)
- Auto-stop functionality at 100% to prevent overruns
- Activity feed showing all cost-related events
- Dismissible alerts with clear all option

### Agent Management
- Add, edit, start/stop, and remove agents
- Support for multiple AI models (GPT-4, Claude, Gemini, etc.)
- Per-agent descriptions and status tracking
- Search and filter functionality

### Professional UI
- Dark and light theme toggle
- Responsive design for mobile and desktop
- Modern SaaS dashboard aesthetic
- Smooth animations and transitions

## How It Works

1. **Add Your Agents**: Register each AI agent with its name, model, and daily budget
2. **Monitor Costs**: Watch real-time cost updates on the dashboard
3. **Set Guardrails**: Configure budget limits and alert thresholds
4. **Receive Alerts**: Get notified when agents approach their limits
5. **Auto-Stop**: Agents automatically stop when they exceed their budget

## Built With
- Vanilla JavaScript (no frameworks - fast and lightweight)
- CSS3 with custom properties for theming
- Chart.js for data visualization
- Local Storage for data persistence
- Novus.ai for user analytics

## Target Audience
- Product Managers overseeing AI-powered features
- Engineering leads managing multiple AI agents
- Startups tracking AI infrastructure costs
- Enterprise teams with budget-constrained AI projects

## Impact
AgentWatch prevents the #1 risk in AI agent deployment: runaway costs. By providing real-time visibility and automatic enforcement, it gives product managers the confidence to deploy AI agents without fear of budget overruns.
