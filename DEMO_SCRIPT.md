# AgentWatch Demo Video Script
# Duration: 2-3 minutes
# Upload to YouTube (unlisted is fine)

## INTRO (15 seconds)
"Hi, I'm presenting AgentWatch - an AI Agent Cost Monitor and Guardrail Dashboard. 
This tool helps product managers track and control AI agent spending in real-time."

## SECTION 1: Dashboard Overview (30 seconds)
- Show the main dashboard
- Point out the 4 stat cards: Total Cost, Active Agents, Budget Alerts, Agents Stopped
- Highlight the cost chart showing 7-day trends
- Show the recent activity feed

## SECTION 2: Agent Management (30 seconds)
- Click on "Agents" in sidebar
- Show the table with 5 demo agents
- Point out cost tracking per agent
- Show budget progress bars (green/yellow/red)
- Demonstrate start/stop functionality

## SECTION 3: Budget Guardrails (30 seconds)
- Click on "Budgets" in sidebar
- Show budget cards for each agent
- Highlight the progress bars showing spending vs limits
- Point out remaining budget calculations

## SECTION 4: Alert System (30 seconds)
- Click on "Alerts" in sidebar
- Show budget warning alerts
- Demonstrate the auto-stop feature when budget exceeded
- Show how alerts are generated in real-time

## SECTION 5: Interactive Features (30 seconds)
- Click "Add Agent" button
- Show the modal form
- Demonstrate adding a new agent
- Show it appears in the dashboard immediately

## SECTION 6: Real-time Simulation (15 seconds)
- Go back to dashboard
- Watch the numbers update automatically
- Show how costs increase over time
- Point out alerts being generated

## SECTION 7: Theme Toggle (15 seconds)
- Click the dark/light mode toggle
- Show the light theme
- Switch back to dark theme

## OUTRO (15 seconds)
"AgentWatch provides real-time visibility into AI agent costs, automatic budget 
enforcement, and instant alerts - helping product managers prevent runaway spending. 
Thank you for watching!"

## RECORDING TIPS
1. Use full screen browser (1920x1080)
2. Zoom to 100% for clear text
3. Move mouse slowly and deliberately
4. Pause briefly on each section
5. Keep total video under 3 minutes
6. Upload to YouTube as "Unlisted"
